# Project Status

Get a quick overview of the current writing project status.

This command will provide:

1. **Project Context**: Summary from CLAUDE.md
   - Project type and topic
   - Current stage
   - Target publication venue

2. **Current Files**: What's in the workspace
   - Latest version(s) in working/
   - Number of archived versions
   - Research materials collected
   - Images prepared

3. **Version History**: Quick summary
   - Current version number
   - How many versions exist
   - Recent changes or milestones

4. **Next Steps**: Suggestions based on current state
   - If drafting: Continue writing, create outline, research
   - If editing: Proofread, revise, get feedback
   - If polishing: Final proofread, prepare for publication
   - If ready: Publish to CMS, archive workspace

5. **Workspace Health**:
   - Any clutter that should be archived?
   - Missing directories needed for workflow?
   - Suggestions for organization

Use this command when:
- You're returning to a project after time away
- You want to understand the current state
- You need to remember what's next
- You want to check if cleanup is needed

Usage: `/status`
